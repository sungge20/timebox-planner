WHY TIMEBOX?
시간을 계획하면
하루가 바뀝니다
타임박스(Time Boxing)는 각 업무에 시작~종료 시간을 미리 정해두는 시간 관리 기법입니다. 마감 없이 일하면 집중력이 흐트러지지만, 시간 경계를 그으면 뇌가 자동으로 집중 모드에 들어갑니다.
⏱
집중력 향상
끝나는 시간을 알면 뇌가 집중합니다. 지금 이 블록만 집중하면 된다는 심리가 생산성을 높입니다.
📊
계획 vs 실제
⭕🔺✕ 상태 표시로 어디서 시간을 낭비하는지 패턴이 보입니다.
🔄
연속적 개선
Yearly→Monthly→Weekly→Daily로 연결되는 구조가 큰 목표를 오늘 일과로 이어줍니다.
탭별 사용 방법
⏱ Daily — 타임박스 출근 후 5분, 퇴근 전 5분
📌 하루 루틴
출근 후 5분
📋 템플릿 불러오기 → Task List 드래그 → 타임박스 배치 → TIME 칸 수정 시 자동 시간순 정렬
퇴근 전 5분
⭕/🔺/✕ 상태 업데이트 → 하단 회고 4칸 작성
Task List
타임박스 위에 이번 달 미완료 Task가 자동 표시됩니다. 📂 지난 일(오늘 이전 미완료) / 📌 오늘 / 🗓 예정(접힘) / 완료(접힘)으로 구분됩니다. 달력에서 날짜 클릭 시 해당 날 기준으로 재분류됩니다.
드래그 배치
Task 카드를 잡고 타임박스 행에 놓으면 Plan 칸에 자동 입력됩니다. 이미 내용이 있으면 교체 여부를 확인합니다.
상태 표시
행 우측 버튼 클릭으로 순환합니다. ⭕ 완료 → Plan 줄긋기 + Monthly Task 완료 처리 / 🔺 진행중 → 주황색 강조 / ✕ 취소 → 흐리게. ⭕ 완료 시 Task List에서 완료 섹션으로 이동합니다.
TIME 입력
TIME 칸을 직접 수정할 수 있습니다. 입력 후 포커스를 벗어나면 시간순으로 자동 정렬됩니다.
📋 템플릿
상단 템플릿 버튼으로 Settings에서 저장한 루틴을 불러옵니다. 현재 타임박스가 교체됩니다.
사이드바
좌측 사이드바 — 달력(날짜 클릭으로 이동) / 통계(블록별 상태 현황) / 습관(오늘 날짜 기준 체크).
자동저장
입력 즉시 브라우저에 자동저장됩니다. 상단에 ✓ 저장됨이 표시됩니다.
📋 Weekly — 주간 플래너 매주 월요일 아침 작성 권장
핵심 목표
이번 주 반드시 끝낼 것 3가지를 먼저 적으세요. 자동저장됩니다.
요일별 일정
각 요일 칸에 중요 미팅·마감·특이사항을 메모하세요. Monthly에서 해당 날짜로 등록한 Task도 표시됩니다. 기본은 월~금, 주말 보기 버튼으로 토·일 추가 가능합니다.
이번 달 목표
Monthly Goals가 자동으로 표시됩니다. ‹ › 버튼으로 이전·다음 주 이동, 오늘 버튼으로 현재 주 복귀.
📅 Monthly — 월간 플래너 매월 1일 작성 권장
달력
Task가 날짜 칸에 색상 칩으로 표시됩니다. 날짜 클릭 시 해당 Daily 탭으로 이동합니다. 칩을 드래그해서 다른 날짜로 이동할 수 있습니다.
Task List
날짜(일)를 숫자로 입력 후 할 일 추가. 텍스트를 직접 클릭해서 수정 가능합니다. Daily Task List와 자동 연동됩니다. Settings에서 등록한 반복 Task는 🔁 섹션에 별도 표시됩니다.
완료 연동
Daily에서 ⭕ 완료 처리하면 Monthly Task도 완료 표시됩니다. 완료 Task는 이후 날짜 Task List에 표시되지 않습니다.
Monthly Goals
이번 달 집중할 목표 5개. Goals 입력 시 Task 데이터에 영향을 주지 않습니다.
🗓 Yearly — 연간 플래너 연초 또는 분기별 작성 권장
연간 목표
기본 3개로 시작, ＋ 목표 추가로 자유롭게 추가합니다. 제목과 메모를 함께 작성하면 동기부여에 좋습니다.
월별 계획
12개월 그리드에 각 달의 테마나 핵심 업무를 간략히 적어두세요.
버킷리스트
올해 꼭 해보고 싶은 것들을 기록하세요. 완료하면 체크합니다.
연간 회고
기본 접힘 상태. 연말에 클릭해서 펼치고 작성하세요.
⚙ Settings
타임박스 설정
시작/종료 시간과 간격(30분·1시간)을 설정합니다. 저장 즉시 Daily 블록에 반영됩니다.
카테고리
카테고리를 추가·삭제하고 색상을 지정할 수 있습니다. 기본 6개 제공(집중·휴식·행정·미팅·피드백·식사).
반복 Task
매주·매달 반복 업무를 등록합니다. 유형(매일·평일·매주·매달), 반복 간격, 시작날짜, 종료조건, 색상 태그를 설정할 수 있습니다. 시간 지정 시 Daily 타임박스에도 자동 추가됩니다. Monthly 탭을 열 때 자동으로 해당 월에 Task가 생성됩니다.
루틴 템플릿
자주 쓰는 하루 블록 구성을 저장합니다. 여러 개 저장 가능, 클릭으로 접기/펼치기. 시간 입력 후 포커스 이동 시 자동 시간순 정렬됩니다. Daily에서 📋 버튼으로 불러옵니다.
데이터 초기화
이번 달 Task·반복 Task·Daily 기록·전체를 선택적으로 삭제할 수 있습니다. 삭제 전 반드시 백업하세요.
💾 데이터 보관 주 1회 이상 백업 권장
모든 데이터는 브라우저 로컬 저장소에 저장됩니다. 브라우저 캐시 삭제·다른 기기 이동 시 데이터가 사라질 수 있습니다.
↓ 백업
상단 버튼 클릭 시 timebox_backup_날짜_시간.json으로 다운로드됩니다.
↑ 불러오기
백업 파일 선택 시 기존 데이터와 병합 복원합니다. 다른 기기에서 이어서 사용할 때 활용하세요.
💡 더 잘 쓰는 팁
연결해서 보기 Yearly → Monthly → Weekly → Daily 순으로 큰 목표가 오늘 일과로 연결됩니다. 방향을 잃었다 싶을 때 Yearly를 다시 열어보세요.
집중 블록 집중 블록 시작 시 슬랙·이메일 알림을 끄세요. 방해 없는 90분이 흩어진 3시간보다 더 많은 일을 해냅니다.
현실적 계획 블록을 빈틈없이 채우지 마세요. 예상치 못한 일을 위해 하루 20~30% 여유 시간을 남겨두세요.
회고 습관 매일 퇴근 전 5분 회고가 가장 큰 변화를 만듭니다. 잘한 점 하나, 개선할 점 하나만 써도 충분합니다.
반복 Task 매주·매달 반드시 하는 업무는 Settings 반복 Task로 등록하세요. 색상 태그로 달력에서 한눈에 구분되고, 시간까지 지정하면 Daily 타임박스에 자동 추가됩니다.
